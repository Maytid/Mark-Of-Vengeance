# Mark Of Vengeance

A fun mod where you can collect bounties from beating levels with the mark of vengeance. A level will receive the mark of vengeance when it has over 5,000 attempts. If the level reaches 5,000 attempts after you already beat it, you have to beat it again to collect the bounty. Both rated and non-rated levels can receive marks of vengeance. Total bounties can be checked in the user profile below your username. You can find a list of levels with those marks of vengeance by checking the suspiciously red box in the Saved tab.

### DISCLAIMER

- This mod is an offline mod which means that it only works with saved levels.
- Other players cannot see your bounty count.
- This mod does not support Safe Mode so if you exploit(noclip, speedhack, etc.), bounties would still be claimed.
- Official levels & levels in the editor cannot receive the mark of vengeance.
- This mod is solely for fun (and maybe motivation).
